# Playwright Automation – ProjeXHRM Deduction Module

Automated UI/E2E tests using **Playwright** (TypeScript) for the **ProjeXHRM** HRM system — specifically the **Deduction Management** page.

## Features Covered
- Employee search by EPF number or name
- Filters: Deduction Type (One Time / Recurring), Category, Effective Date Range
- Validation of deduction entries (amount, reason, effective/start/end dates, payment terms)
- Handling of UI elements: dropdowns, date pickers, search buttons, reset filters

## Tech Stack
- Playwright (latest version)
- TypeScript
- Node.js
